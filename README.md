# 6DPoseDataGenerator
6D Pose Training Data Generator via Unity for EfficientPose

photo-realistic한 객체를 이용하여 현실과 유사하면서도, 무작위한 객체 위치/회전을 생성하여 다양한 모양의 학습 데이터를 생성한다.

saving files:
 - camera intrinsic matrix
 - rgb image
 - binary mask image
 - 3x3 rotation matrix
 - 3x1 translation vector
 - 2D center point
 - object label
